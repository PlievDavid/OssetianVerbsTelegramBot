# Telegram-бот для изучения осетинского языка.
Презентация: https://docs.google.com/presentation/d/1aaOQV-GCOWlnCFjDTMzKlQXngOA3kkIM7vm5rvljzzc/edit?usp=sharing
## Об осетинских глаголах.
В осетинском языке глаголы делятся на два типа. Тип глагола определяет правила его склонения. Пример:
![oset-verbs-types](https://github.com/user-attachments/assets/3735e853-ee54-46d0-9a88-b96b72a2dadc)
Часто при склонении меняетя основа инфинитива глагола. Например: дз***у***рын - дз***ы***рдтон.
## Виды заданий.
### 1. Выбор правильного окончания.
Даётся глагол на осетинском и два варианта его окончания. Необходимо выбрать правильный вариант с учетом типа глагола.
### 2. Перевод с русского на осетинский.
Даётся простое предложение на русском языке(предложение вида: подлежащее + сказуемое) и два варианта его перевода на осетинский. Необходимо выбрать правильный вариант с учетом типа глагола.
### 3. Определение типа глагола.
Даётся глагол на осетинском. Необходимо выбрать, к какому типу он относится(1 или 2).
## Взаимодействе с пользователем
Тест будет содержать 10 заданий, вид которых будет определяться случайно.
После прохождения теста пользователю будет выдано количество правильных ответов.
Будет собираться статистика слов, в которых пользователь чаще всего ошибался.
## Реализация
### База данных
Будет использоваться база данных SQLite. Одна таблица будет хранить глаголы, их перевод и тип(цифра 1 или 2). Другая таблица будет хранить статистику ошибок пользователя(глагол и сколько раз в нем ошибся пользователь). Также будет json или xml файл, содержащий правила склонения глагола по типу(нужен, чтобы не хранить все формы склонения глагола).
